Html Tag

<html> tag is the starting section of the html document. Every other sections are inside html tag only. It defines the meaning and structure of web content.

Body Tag

<bold> tag is another tag that is one level down html tag. The main document to be shown is written inside body.

Font Styles

We have different tags to style the elements of the document.

<b> is used to bolden the text inside the tags.

<u> is used to underline the text inside the tags.

<i> is used to italic the text inside the tags.

Table

<table> tag is used to create tables.

  • <tr> - To create table rows.
  • <td> - To create table data.

Code


        <table>
        <tr>
          <td>ID</td>
          <td>Name</td>
        </tr>
        <tr>
          <td>39529</td>
          <td>Gourab Porey</td>
        </tr>
        <tr>
          <td>39532</td>
          <td>Milan Chakraborty</td>
        </tr>
      </table>

Output:

ID Name
39529 Gourab Porey
39532 Milan Chakraborty

Anchor Tag

The <a> element create a hyperlink to web pages, files, inpage etc which a URL can address.

Content with each a element should indicate the target address. If the href attribute is set properly pressing the enter key while focused on the <a> element will activate it. 


      <a href="https://google.com">Visit Google</a>
Visit Google

To open the link in new page, we can specify
target _blank: usually a new tab, but users can configure browsers to open a new window instead. 


      <a href="https://google.com" target="_blank">Visit Google</a>
Visit Google

img Tag

The img element embed an image into the html page.

It has attribute src that contains the path to the image to be embedded.
